# Menu

## Import

```jsx
import { Menu } from 'bumbag-native';
```

## Usage

Supply a set of `<Menu.Item>` components as children to `<Menu>` to create a menu.

```jsx-live
<Box backgroundColor="white700" padding="major-2">
  <Menu>
    <Menu.Item>
      General
    </Menu.Item>
    <Menu.Item>
      Accounts
    </Menu.Item>
    <Menu.Item>
      Alerts
    </Menu.Item>
    <Menu.Item>
      Security
    </Menu.Item>
  </Menu>
</Box>
```

### Dividers

To create dividers in your menu, use the `hasDividers` prop.

```jsx-live
<Box backgroundColor="white700" padding="major-2">
  <Menu hasDividers>
    <Menu.Item>
      General
    </Menu.Item>
    <Menu.Item>
      Accounts
    </Menu.Item>
    <Menu.Item>
      Alerts
    </Menu.Item>
    <Menu.Item>
      Security
    </Menu.Item>
  </Menu>
</Box>
```
